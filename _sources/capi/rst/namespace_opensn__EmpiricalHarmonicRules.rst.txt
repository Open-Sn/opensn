Namespace opensn::EmpiricalHarmonicRules
========================================


Empirical harmonic selection rules for special quadrature types. 




.. contents:: Contents
   :local:

Detailed Description
--------------------

This namespace contains pre-computed empirical rules for quadrature types where mathematical rules are not easily derived or where empirical selection provides better results.
