Starting latch.

Sweep operations wait on this latch to be fully released before starting. The OS puts the thread to sleep while waiting, avoiding taking up CPU resources for spinning wait.

std::vector<AAHD_AngleSet*> following_angle_sets_

List of AAHD_AngleSets waiting after this angle set.
