This is a vector [dependent_location][unordered_cell_index] that holds an AlphaPair. AlphaPair-first is the cell’s global_id and AlphaPair-second holds a number of BetaPairs. Each BetaPair comprises BetaPair-first = face_slot_index (the location of this faces data in the psi vector, and then a vector of vertex indexes that can be used for dof_mapping. Filled during slot-dynamics. Cleared after beta-pass.
