Define OpenSnRegisterObjectAliasInNamespace

  • Defined in File object_factory.h

Define Documentation

OpenSnRegisterObjectAliasInNamespace(namespace_name, alias, object_name)

Macro for registering an object alias within the ObjectFactory.

Note

This will register a C++ class object_name such that it will show up as namespace_name.alias

Parameters:

  • namespace_name – Namespace name

  • alias – Name of the object

  • object_name – C++ class name to register.
