============
Installation
============

The user guide does not duplicate the installation instructions.

For current installation and build instructions, see the maintained install
documentation in:

* :doc:`/install/index`

That section covers:

* required development tools and third-party packages,
* dependency installation,
* building the console application and Python module,
* running regression tests, and
* building the documentation.

.. note::

   Keeping the detailed installation steps in one place helps avoid drift
   between the user guide and the maintained build instructions.
